# Nombre del curso: Microsoft Project

## Descripción del curso:
Este curso está diseñado para orientar el conocimiento y práctica del participante en el Software de especialidad para la Dirección de Proyectos Microsoft Project 2019 Professional. Durante el desarrollo de esta disciplina, se espera que el participante aprenda a usar y aprovechar la herramienta para realizar la planificación detallada de cada etapa del proyecto, así como dar seguimiento a las tareas durante su ejecución; haciendo hincapié en las facilidades que ofrece en materia de generación de reportes, control de tareas hasta su terminación, el uso de recursos, el costo del proyecto y, sobre todo, la comunicación efectiva entre los involucrados en el proyecto.
